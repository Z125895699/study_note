潘玮柏将结婚 对象是上海空姐?工作人员保持沉默

网曝潘玮柏即将结婚，网友猜测潘玮柏结婚对象是曾传过绯闻的上海小姐姐空姐。新浪娱乐就此求潘玮柏工作室，对方保持沉默，暂无回应




潘玮柏绯闻女友

    新浪娱乐讯近日，网曝潘玮柏[微博]父亲在台湾某学校讲课，其中提到了潘玮柏，无意之间向同学们爆料潘玮柏就即将结婚。不少网友爆料称潘玮柏结婚对象是曾传过绯闻的上海空姐，新浪娱乐就此求证潘玮柏工作室，工作人员保持沉默，暂无回应。

    潘玮柏爸爸潘志伟是台湾实践大学、淡江大学企业管理专业的教授，而这次网友爆料的同时还晒出了潘爸爸的正面照。还有网友猜测潘玮柏的结婚对象是之前的绯闻女友上海空姐，两个人不仅有很多情侣同款，还曾传出已同居的消息。（新娱/文）
